## 📖 About Shooter's Eye

Shooter's Eye is a Windows desktop application that provides:
- Automated shot detection and timing analysis
- Frame-by-frame video playback and inspection
- Drawing and annotation tools for coaching feedback
- Side-by-side comparison of multiple runs
- Video export with analysis overlays

For more information, visit the main repository: https://github.com/rt-ghost6/ShootersEye
